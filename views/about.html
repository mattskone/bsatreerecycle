So, what's this all about...?

Many Boy Scout Troops across the country raise money for Scouting activities by collecting and recycling Christmas trees in their community for a nominal donation. During these fundraisers, adult drivers can use this website via their smartphones to mark the location of each tree pickup quickly and easily. The event coordinator can monitor each driver's progress as tree pickups are updated real-time on the map.

How do I use this site?

If you're a driver, you probably have a car full of Scouts, a trailer hitched to your car and a map of the neighborhoods you're supposed to cover. Pop some Excedrin and grab another cup of coffee. Next, navigate to the Mark a Pickup page on your smartphone and set it next to you in the car. Each time you stop to pick up a tree, hit the "Mark a Pickup" button, and the website will record the location and time of the pickup. You can tag your pickups to identify them as yours - just use the "Change Tag" link to change your tag to something like "Mike's Car", or "Raven Patrol".

The only tricky part is "enabling location services" on your phone, which tells your phone it's ok to give away your location information. Many folks turn off location services out of privacy concerns, and for good reason. However, it must be turned on if you're going to use this website. If you need help, here are some articles for various types of smartphone operating systems:

  • Android
  • iOS 6
  • iOS 5
  • iOS 4
  • Windows Phone 8
  • Windows Phone 7

If you're the event coordinator, use the map to zoom in on your community. As drivers mark each tree pickup, a small tree icon will appear on the map. If you hover over a tree icon with your mouse, you'll see the date and time of that pickup, and you'll see the tag, such as "Mike's Car" or "Raven Patrol", that your driver is using (see above).


<Geeky stuff \>

This site is hosted by Heroku, and is built on node.js using the express web framework. Data is stored in a MongoDB database hosted by MongoHQ. Pages are built with the help of Twitter's outstanding Bootstrap framework and rendered using Handlebars templates. The whole thing is written in javascript and HTML5, and the source code is (almost) available on GitHub.
